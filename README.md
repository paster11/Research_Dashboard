# Research Intelligence Dashboard

Cross-provider investment research summary — BlackRock Investment Institute (weekly), Invesco Solutions TAA (monthly), Morgan Stanley "The BEAT" (quarterly).

Live dashboard: open `index.html`, or view via GitHub Pages once enabled.

## Deploy on GitHub Pages

1. Create a new repository on GitHub (e.g. `research-dashboard`).
2. Upload both files in this folder (`index.html`, `README.md`).
3. Go to **Settings → Pages → Source: Deploy from a branch → main / (root) → Save**.
4. The dashboard will be live at `https://<username>.github.io/research-dashboard/` within a minute or two.

## Updating

Replace `index.html` with the latest dashboard version and commit — Pages redeploys automatically on every push.

## Note

Content is an interpretive summary of research publications intended for professional/institutional investors. Intended for internal team use — verify against original documents before acting, and consider access restrictions before sharing the URL.
